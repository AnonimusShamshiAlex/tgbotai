# Telegram AI Bot

Telegram бот с искусственным интеллектом. Понимает текстовые и голосовые сообщения, отвечает на русском языке через OpenRouter API.

OpenRouter ключ:

Зарегистрируйся на https://openrouter.ai/settings/keys

Перейди в Keys

Нажми "Create Key"

Скопируй ключ (начинается с sk-or-v1-)

## Модел

https://openrouter.ai/stepfun/step-3.5-flash:free

## Botfather 

https://t.me/BotFather - Создание бота в Telegram выполняется через официального бота
@BotFather.


## Возможности
- Текстовые сообщения
- Голосовые сообщения (распознавание речи)
- Бесплатные AI модели
- Русский язык
